# TechStore - Tienda de Teléfonos

Este es un sitio web responsive para una tienda de teléfonos con diseño similar a Apple.com, desarrollado para ser desplegado en GitHub Pages.

## Características

- Diseño responsive similar a Apple.com
- Paleta de colores personalizada:
  - Fondo: #F5F5F7
  - Texto: #1D1D1F
  - Bordes y acentos: #0276DF
  - Texto en menús: #FFFFFF
- Cuatro páginas interconectadas:
  1. Página principal con menú y categorías
  2. Página de iPhone nuevos
  3. Página de iPhone refurbished
  4. Página de Samsung nuevos
- Iconos de batería para productos refurbished
- Círculos de colores para representar los colores de iPhone
- Precios formateados en pesos colombianos (ej: $2.450.000)
